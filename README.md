# 🚀 AI-Powered Interview Preparation Platform

An intelligent full-stack web application that helps users prepare for job interviews by analyzing their **resume** and **target job description**, identifying **skill gaps**, and generating a **personalized interview preparation plan** using AI.

---

## 🌟 Features

### 🔐 User Authentication
- Secure Login & Registration
- JWT-based authentication

### 📄 Resume Upload & Analysis
- Upload Resume (PDF/DOCX)
- AI extracts skills from resume

### 🧠 Job Description Analysis
- Paste target job description
- Extracts required skills and expectations

### 📊 Skill Gap Detection
- Compares Resume vs Job Description
- Highlights missing skills

### ❓ AI-Generated Questions
- Technical Questions
- Behavioral Questions

### 🛣️ Personalized Preparation Roadmap
- Day-wise plan (5-day structured roadmap)
- Focused learning based on skill gaps

### 📄 ATS-Friendly Resume Generator
- AI-generated optimized resume
- Downloadable PDF
